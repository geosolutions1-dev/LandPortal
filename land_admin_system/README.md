# Land Service Portal

A comprehensive land administration system for Ghana.

## Features
- User registration and authentication
- Appointment booking for land services
- Document upload and management
- Online payment integration
- Admin dashboard for processing applications
- Communication tracking (Email, WhatsApp, Phone, Zoom)

## Deployment on Render

### Prerequisites
- GitHub account
- Render account (free tier available)
